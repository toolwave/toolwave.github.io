    如何切换Duolingo学习的目标语言

    1. 首先，我们初次进入Duolingo时，需要选择一种要学习的语言，这里我们以"中文"为例。
    2. 然后我们需要做一个单元测试，这个过程不会很长，大概15个基本问题。
    3. 完成一些基础的测试之后，我们就要正式进入语言学习计划了。
    4. 这时选择左上角的语言课程按钮，可以切换到其他语言进行学习。

    如何切换 Duolingo App 的语言

    • Duolingo App的界面语言会根据用户手机的系统语言自动识别。
    • 我们需要更改用户手机的操作系统语言。
    • 然后重新启动程序，它就会自动切换到你想要的语言界面。

    Android系统语言切换方法

    [步骤] ：设置->系统->语言和输入->语言

    1. 通过手机设置进入设置菜单。
    2. 在系统设置中，找到 语言和输入 选项。
    3. 你可以在这里修改系统语言。
    4. 重新启动 Duolingo 以重新加载。

    iOS系统语言切换方法

    [步骤] ：设置->常规->语言和地区

    1. 首先找到IOS系统里的设置。
    2. 然后在菜单中找到 【常规】 。
    3. 在菜单列表中，找到 [语言和地区] 。
    4. 你可以在这里修改IOS的系统语言。
    5. 然后重启Duolingo即可修改App界面语言。

    如何删除 Duolingo 中的语言课程

    1. 首先我们进入Duolingo的主界面，选择个人资料。
    2. 选择右上角的设置，在菜单列表中找到 【管理课程】 。
    3. 你可以在这里直接删除你不想学习的语言课程。

    注意：删除课程后，你将无法恢复该课程，并且所有学习进度都将丢失。请谨慎操作。
