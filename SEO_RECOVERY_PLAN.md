## 🔧 具体操作步骤

### 如何批量更新日期？

**方法1：使用脚本批量替换**
创建一个脚本文件 `update-dates.js`：

```javascript
// 遍历所有HTML文件
const fs = require('fs');
const files = fs.readdirSync('./').filter(f => f.endsWith('.html'));

files.forEach(file => {
  let content = fs.readFileSync(file, 'utf8');
  // 更新 dateModified
  content = content.replace(
    /"dateModified":\s*"\d{4}-\d{2}-\d{2}"/g,
    '"dateModified": "2026-01-12"'
  );
  fs.writeFileSync(file, content);
});
```

**方法2：使用VS Code批量替换**
1. 打开VS Code
2. Ctrl+Shift+F（全局搜索）
3. 搜索：`"dateModified": "2025-`
4. 替换为：`"dateModified": "2026-01-12"`
5. 点击"全部替换"

### 如何301重定向？

由于是GitHub Pages，你需要：
1. 在要重定向的HTML文件顶部添加：
```html
<!DOCTYPE html>
<html>
<head>
  <link rel="canonical" href="https://toolwave.github.io/chatgpt-5.html">
  <meta http-equiv="refresh" content="0;url=https://toolwave.github.io/chatgpt-5.html">
  <script>window.location.href='https://toolwave.github.io/chatgpt-5.html';</script>
</head>
<body></body>
</html>
```

2. 更新 sitemap-zh.xml，移除重定向的URL

3. 在GSC提交这些地址变更
