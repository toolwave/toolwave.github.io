    隐私政策

    最后更新：2026年1月13日

    网站性质

    ToolWave 是一个托管在 GitHub Pages 上的静态网站，由纯 HTML、CSS 和 JavaScript 构成。没有后端服务器、没有数据库、没有用户系统。

    数据收集

    本网站不收集、存储或处理任何用户个人信息，包括：

    • 不要求用户注册或登录
    • 不使用 Cookie 或其他追踪技术
    • 不使用数据分析工具（如 Google Analytics）
    • 不收集 IP 地址、设备信息或浏览行为

    第三方链接

    本网站可能包含指向其他网站的链接（如流媒体服务官方网站）。一旦您离开本网站，这些第三方网站的隐私政策将适用。

    建议您在访问任何第三方网站前查看其隐私政策。

    GitHub 日志

    作为托管平台，GitHub 可能会记录基本的服务器日志（如访问时间、请求的页面）。这些日志由 GitHub 管理，我无法访问。详情请参考 GitHub 隐私声明。

    儿童隐私

    本网站适合所有年龄段观众。由于不收集任何信息，不存在儿童个人信息保护的问题。

    政策更新

    我可能会不时更新本隐私政策。重大变更时会在网站首页发布通知。
